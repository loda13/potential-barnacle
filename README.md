# 股票K线技术分析工具

命令行股票技术分析工具，支持A股/美股/港股，提供多指标共振分析、资金流向、新闻舆情、买卖点决策等完整分析体系。

## 安装

```bash
pip3 install pandas numpy yfinance akshare
```

> 注：已支持纯pandas/numpy实现，无需安装pandas_ta

## 使用

```bash
# A股日线（贵州茅台，最近60天）
python3 stock_analyzer.py 600519 -d 60

# 美股周线（苹果，获取约34周数据）
python3 stock_analyzer.py AAPL -p w -d 240

# 港股（腾讯控股）
python3 stock_analyzer.py 0700.HK -d 60

# 演示模式（离线，无需网络）
python3 stock_analyzer.py 600519 --demo
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `code` | 股票代码（A股6位数字，美股如AAPL，港股如0700.HK） | 必填 |
| `-p, --period` | K线周期：`d`=日线，`w`=周线 | `d` |
| `-d, --days` | 分析天数/周数（周线建议≥240） | `60` |
| `--demo` | 演示模式，使用模拟数据 | 关闭 |

## 功能特性

### 核心技术指标
- **均线系统**: MA5/10/20 多头/空头排列判断
- **RSI**: 相对强弱指标，超买(>70)/超卖(<30)
- **MACD**: 异同移动平均线，金叉/死叉信号
- **布林带**: 突破上下轨信号
- **KDJ**: 随机指标，J值超买超卖
- **ADX**: 平均趋向指数，趋势强度判断
- **ATR**: 平均真实波幅，波动性/止损参考
- **OBV**: 能量潮，资金流向判断
- **CCI**: 顺势指标，超买超卖
- **SuperTrend**: 超级趋势，趋势跟踪
- **PSAR**: 抛物线转向，止损位
- **Ichimoku**: 一目均衡表，云图支撑压力

### 新增高级功能

| 功能 | 说明 | 数据源 |
|------|------|--------|
| 筹码分布 | 集中度、主力成本区、浮筹比例 | akshare |
| 主力资金流 | 当日/5日净流入、量比、资金动向 | akshare/yfinance |
| 市场舆情 | 最近5条新闻、情绪分析 | yfinance |
| 大盘复盘 | 主要指数涨跌幅、热门板块 | yfinance/akshare |
| 精确买卖点 | 买入价/止损价/目标价、风险报酬比 | 指标计算 |
| 操作检查清单 | 10项买入条件检查 | 指标计算 |
| LLM决策仪表盘 | 综合评分、行动建议 | 规则评分 |

## 共振信号说明

| 共振信号 | 条件 | 含义 |
|----------|------|------|
| ▲▲ 强烈买入 | 买入指标占比≥70% | 多指标强烈看多 |
| ▲ 买入 | 买入指标占比≥50% | 偏多信号 |
| ○ 多空分歧 | 买卖指标均未过半 | 方向不明 |
| ▼ 卖出 | 卖出指标占比≥50% | 偏空信号 |
| ▼▼ 强烈卖出 | 卖出指标占比≥70% | 多指标强烈看空 |

## LLM决策仪表盘评分体系

| 维度 | 权重 | 评分说明 |
|------|------|----------|
| 技术面 | 40% | strong_buy=90, buy=70, neutral=50, sell=30, strong_sell=10 |
| 资金面 | 30% | 持续流入=80, 净流入=65, 观望=50, 净流出=35, 持续流出=20 |
| 舆情 | 15% | 利好=80, 中性=50, 利空=20 |
| 机构 | 15% | buy=85, hold=50, sell=15 |

## 注意事项

1. **数据源**：A股优先使用akshare，美股/港股使用yfinance
2. **API限速**：yfinance有请求频率限制，频繁请求可能被临时封禁
3. **数据要求**：至少需要100条数据才能完整计算所有指标
4. **免责声明**：本工具仅供参考，不构成投资建议

## 常见问题

**Q: 提示"无法获取股票数据"？**
A: 检查网络连接，或使用`--demo`模式测试

**Q: A股代码如何输入？**
A: 直接输入6位数字即可，如`600519`、`000001`

**Q: 港股如何输入？**
A: 使用yfinance格式，如`0700.HK`（腾讯）、`9999.HK`（网易）

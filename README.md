# 股票K线技术分析工具

命令行股票技术分析工具，支持A股/美股/港股，提供多指标共振分析、资金流向、新闻舆情、买卖点决策等完整分析体系。

**🆕 v3.0 机构级多策略交易系统（全周期分析框架）**：
- **左侧埋伏策略**：均值回归Z-Score、量能衰竭、波动率状态、机构级支撑压力、三重背离检测
- **右侧确认策略**：Break of Structure (BoS)、Change of Character (ChoCh) 趋势跟随信号
- **机构级退出策略**：VWAP 偏离度分析、高位派发模式识别（放量滞涨、顶背离、破位下跌）
- **批量分析**：支持同时分析多只股票，生成多空力量对比表
- **压力测试**：15% 回撤场景分析、VaR 计算、智能对冲策略建议

核心K线行情默认使用 Yahoo chart API；新闻、评级、筹码、大盘复盘等扩展信息会在失败时自动降级，不会中断主分析流程。

## 安装

```bash
pip3 install -r requirements.txt
```

> 注：已支持纯pandas/numpy实现，无需安装pandas_ta

## 使用

```bash
# A股日线（贵州茅台，最近60天）
python3 stock_analyzer.py 600519 -d 60

# 美股周线（苹果，获取约34周数据）
python3 stock_analyzer.py AAPL -p w -d 240

# 港股（腾讯控股）
python3 stock_analyzer.py 0700.HK -d 60

# 港股（小米集团）
python3 stock_analyzer.py 1810.HK -d 60

# 美股（特斯拉 / 英伟达）
python3 stock_analyzer.py TSLA -d 60
python3 stock_analyzer.py NVDA -d 60

# 🆕 批量分析多只股票（生成对比表）
python3 stock_analyzer.py --portfolio 0700.HK 1810.HK QQQ TSLA NVDA -d 60

# 🆕 单股票 + 压力测试（15% 回撤场景）
python3 stock_analyzer.py NVDA -d 60 --stress-test

# 🆕 批量分析 + 压力测试
python3 stock_analyzer.py --portfolio QQQ TSLA NVDA -d 60 --stress-test

# 演示模式（离线，不拉取扩展网络数据）
python3 stock_analyzer.py TSLA --demo
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `code` | 股票代码（A股6位数字，美股如AAPL，港股如0700.HK） | 必填（单股票模式） |
| `-p, --period` | K线周期：`d`=日线，`w`=周线 | `d` |
| `-d, --days` | 分析天数/周数（周线建议≥240） | `60` |
| `--demo` | 演示模式，使用模拟数据 | 关闭 |
| `--portfolio` | 批量分析多只股票（例：`--portfolio 0700.HK QQQ TSLA`） | 无 |
| `--stress-test` | 执行 15% 回撤压力测试 | 关闭 |

## 运行说明

- 核心分析成功的判定标准是：命令可以输出完整技术分析结果，即使部分扩展数据源返回“暂无数据”或“获取失败”也不会中断。
- `--demo` 模式只使用本地模拟K线数据，并跳过大盘、新闻、评级、筹码等联网扩展数据。
- A股在 Yahoo 行情失败时会自动尝试 `akshare` 作为备选。

## 功能特性

### 核心技术指标
- **均线系统**: MA5/10/20 多头/空头排列判断
- **RSI**: 相对强弱指标，超买(>70)/超卖(<30)
- **MACD**: 异同移动平均线，金叉/死叉信号
- **布林带**: 突破上下轨信号
- **KDJ**: 随机指标，J值超买超卖
- **ADX**: 平均趋向指数，趋势强度判断
- **ATR**: 平均真实波幅，波动性/止损参考
- **OBV**: 能量潮，资金流向判断
- **CCI**: 顺势指标，超买超卖
- **SuperTrend**: 超级趋势，趋势跟踪
- **PSAR**: 抛物线转向，止损位
- **Ichimoku**: 一目均衡表，云图支撑压力

### 新增高级功能

| 功能 | 说明 | 数据源 |
|------|------|--------|
| 筹码分布 | 集中度、主力成本区、浮筹比例 | akshare |
| 主力资金流 | 当日/5日净流入、量比、资金动向 | akshare/技术分析 |
| 市场舆情 | 最近5条新闻、情绪分析 | yfinance |
| 大盘复盘 | 主要指数涨跌幅、热门板块 | Yahoo chart API/akshare |
| 精确买卖点 | 买入价/止损价/目标价、风险报酬比 | 指标计算 |
| 操作检查清单 | 12项买入条件检查 | 指标计算 |
| LLM决策仪表盘 | 综合评分、行动建议 | 规则评分 |

### 🆕 左侧交易分析（机构级逆势抄底）

| 模块 | 说明 | 核心指标 |
|------|------|----------|
| 均值回归Z-Score | 识别极端超卖区域 | 价格偏离200日均线标准差，Z<-2.0为超卖 |
| 量能衰竭分析 | 检测转折点 | 放量滞跌、缩量阴跌、高量节点(HVN) |
| 波动率收缩/扩张 | 捕捉脉冲信号 | ATR百分位、布林带挤压、首次扩张 |
| 机构级支撑压力 | 精准支撑位 | Order Blocks、Fair Value Gaps(FVG)、假破位 |
| 三重背离检测 | 量价背离 | RSI/OBV/MACD柱状图与价格背离 |
| 左侧信号聚合 | 综合评分0-100 | 加权合成5模块，生成仓位建议 |

**仓位管理策略**：
- 左侧试探仓（5-10%）：在超卖区域建立初始仓位
- 右侧确认加仓（15-20%）：等待MACD金叉、站上均线等确认信号
- 风险警告：自动检测趋势性下跌、恐慌抛售等失效场景

### 🆕 右侧确认信号（趋势跟随）

| 模块 | 说明 | 核心指标 |
|------|------|----------|
| Break of Structure (BoS) | 识别价格突破关键波段 | 突破前一个波段高点（看多）或低点（看空） |
| Change of Character (ChoCh) | 识别趋势性质改变 | 上升趋势中出现 Lower Low（看空转折）<br>下降趋势中出现 Higher High（看多转折） |
| 趋势识别 | 基于 ADX + MA 判断 | ADX > 25 且 MA5 > MA20 = 上升趋势 |
| 成交量确认 | 突破有效性验证 | 成交量 > MA20_volume |

**信号强度**：
- BoS 强度：基于突破幅度计算（0-100）
- ChoCh 强度：基于反向波段幅度 / ATR 标准化（0-100）
- 确认条件：强度 > 50 且成交量放大

### 🆕 机构级退出策略

| 模块 | 说明 | 触发条件 |
|------|------|----------|
| VWAP 偏离度分析 | 机构级止盈触发器 | 价格偏离 VWAP > 3σ：强制减仓<br>价格偏离 VWAP > 2σ：部分止盈 |
| 高位派发识别 | 检测机构退出信号 | 放量滞涨：成交量创新高但价格不涨<br>顶背离：价格新高但 RSI/MACD 不创新高<br>破位下跌：跌破 MA20 且 MACD 死叉 |
| 派发风险评分 | 综合评估 0-100 | ≥70 分：强烈派发信号，建议减仓/清仓<br>≥40 分：出现派发迹象，密切关注 |

### 🆕 批量分析与压力测试

| 功能 | 说明 | 输出内容 |
|------|------|----------|
| 多股票对比表 | 横向比较股票强弱 | 当前阶段、多空强度、共振信号、BoS/ChoCh 状态、派发风险、建议操作 |
| 市场阶段判断 | 自动识别所处阶段 | 左侧筑底、右侧启动、高位派发、下行通道、震荡整理 |
| 压力测试 | 15% 回撤场景分析 | VaR (95% 置信度)、下方支撑位、对冲策略建议 |
| 对冲工具推荐 | 根据股票类型智能推荐 | 美股科技股：VIX 期权、个股期权、纳指期货<br>港股：恒指期货、恒指期权、美元资产<br>A股：沪深 300/中证 500 期货、期权、国债 |

## LLM决策仪表盘评分体系

| 维度 | 权重 | 评分说明 |
|------|------|----------|
| 技术面 | 30% | strong_buy=90, buy=70, neutral=50, sell=30, strong_sell=10 |
| 资金面 | 25% | 持续流入=80, 净流入=65, 观望=50, 净流出=35, 持续流出=20 |
| 舆情 | 10% | 利好=80, 中性=50, 利空=20 |
| 机构 | 10% | buy=85, hold=50, sell=15 |
| 左侧信号 | 25% | 来自左侧综合评分(0-100) |

**行动建议映射**：
- 综合评分 ≥ 75：【加仓】（高置信度）
- 综合评分 60-74：【买入】（中置信度）
- 综合评分 40-59：【持仓观望】（低置信度）
- 综合评分 25-39：【减仓】（中置信度）
- 综合评分 < 25：【清仓】（高置信度）

## 注意事项

1. **数据源**：A股优先使用akshare，美股/港股使用yfinance
2. **API限速**：yfinance有请求频率限制，频繁请求可能被临时封禁
3. **数据要求**：
   - 基础指标：至少需要100条数据
   - 左侧交易分析：建议使用 `-d 120` 或更多，以获取200日均线数据
   - 批量分析：建议使用演示模式 `--demo` 避免 API 限速
4. **免责声明**：本工具仅供参考，不构成投资建议

## 常见问题

**Q: 提示"无法获取股票数据"？**
A: 检查网络连接，或使用`--demo`模式测试

**Q: A股代码如何输入？**
A: 直接输入6位数字即可，如`600519`、`000001`

**Q: 港股如何输入？**
A: 使用yfinance格式，如`0700.HK`（腾讯）、`9999.HK`（网易）
